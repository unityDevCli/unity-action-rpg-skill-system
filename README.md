# Unity Action RPG Skill System

A focused Unity demo showcasing a **data-driven skill system**
designed for **Action RPG / Survivor-like** games.

## Features
- Active & passive skills
- Cooldown & resource management
- Buff / debuff with duration & stacking
- Modular skill execution pipeline

## Architecture Highlights
- Data-driven using ScriptableObject
- Event-based communication
- Clean separation between data, logic and presentation
- Easily extendable without modifying existing code

## Tech Stack
- Unity 2022+
- C#
- ScriptableObject
- Event-based architecture

## How to Run
1. Open project with Unity 2022+
2. Load `DemoScene`
3. Press Play and use keyboard to trigger skills
